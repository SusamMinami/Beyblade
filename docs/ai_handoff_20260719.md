## 组装交互

- 模型尺寸根据默认相机下的屏幕投影动态拟合。
- 默认不选择部件，只显示当前陀螺和弱化的相邻陀螺。
- 点击模型：射线检测具体部件并展开对应的三个零件选项。
- 选中部件保持实体，其他四个部件降为 20% 透明度。
- 点击空白区域退出改装并恢复全部实体显示。
- 在模型或背景上拖动都环绕相机，不再单独旋转模型。
- 空白区域横扫或顶部左右触控区可切换三个独立出战槽位。
- 滚轮或双指：连续缩放。
- 无操作时当前陀螺缓慢自旋，相机做小幅待机漂移。
- 切换出战槽时复用现有模型，让新陀螺从侧面滚动到中央。
- 点选不同部位时相机平滑聚焦对应高度，可在编辑状态直接切换其他部位。
- 换件使用约 300ms 的拆出/装入动画，不重置当前镜头。

浏览器已验证部件聚焦、点击其他层直接切换、点击空白退出、统一相机旋转和滚动换陀螺。

## 地图、发射与战斗 HUD

- 地图页使用横向 `scroll-snap` 卡片，当前居中卡片与 Three.js 场地同步。
- 地图页底部只有“准备发射”一个主按钮。
- 战斗准备阶段显示程序化 3D 发射器。
- 发射准备不显示参数滑杆窗口。拖动模型直接改变方向与倾角；拖动黄色箭头端点改变
  方向和力度，箭头长度代表力度。
- 发射器黄色导轨已抬高，避免与本体重叠产生深度闪烁。
- 发射后摇杆固定在屏幕底部安全区；陀螺附近只显示控制影响和施力方向。
- 战斗时普通品牌、阶段、金币和声音导航隐藏。双方转速、耐久和时间集中到顶栏。
- 标准地面标签与碰撞/出界/转速/倾角状态窗已移除；风险继续用场内效果和音效反馈。

## Web 失衡、战斗反馈与零件 DIY 更新

本轮优先继续验证 Web 玩法，新增：

- `BattleSimulation` 碰撞诊断日志，记录双方碰撞前后的倾角、转速、失衡、
  耐久和变化量。构造参数 `diagnostics` 控制是否输出到 `logger`，
  `collisionLog` 最多保留最近 200 条。
- 确定性失衡值 `imbalance`。高冲量、低稳定、低转速和不利地形会放大失衡；
  失衡会抬高倾角、削弱控制，严重倾斜会擦地并额外损失转速。
- 地形的 `stability` 参数正式进入倾角与失衡恢复计算。
- 玩家状态增加 `spinLossRate`、`ringOutRisk`、`stabilityState`、
  `ringRiskState` 和 `spinRiskState`。
- 碰撞、出界、转速和倾斜风险保留确定性状态与分级音效，但不再显示遮挡场地的窗口。
- 风险等级变化触发短促音效和有限振动；重碰撞触发冲击环和短暂画面震动。
- 摇杆固定在屏幕底部，不再跟随高速移动的陀螺。陀螺附近只显示控制影响百分比、
  施力箭头和影响范围环。
- 再次点击当前已装备零件可进入深度 DIY。蓝色、橙色、红色模型手柄分别直接调整
  整体大小、部件高度和轮廓起伏，不再使用底部参数滑杆。
- DIY 顶栏提供正视、俯视、侧视、实时质量/惯量和 2 / 3 / 4 / 6 边对称延展；
  切换对称方式只改变延展规则，不重置已有大小、高度、轮廓或材料。
- 原装/聚合物/合金/碳纤维/橡胶材料通过按钮选择，非原装材料使用金币永久解锁，
  不再提供重复的材料滑杆。
- DIY 底栏只保留材料列表、保存和取消。质心仅在 3D 模型上显示，不再重复显示
  XYZ 文本窗口。
- DIY 草稿实时重建程序化模型并更新顶栏质量/惯量；保存写入当前出战槽，取消恢复原值。
- `AssemblyCalculator` 使用改造后的零件参数重新计算质量、质心和惯量；绕自旋轴
  惯量包含平行轴项。
- 对战页使用粗黑描边、错位阴影、纸片式读数、放射速度线和夸张标题强化涂鸦感，
  但中央竞技场仍保持无遮挡。

## Web 交互连贯性与胜利反馈更新

本轮继续收敛 Web 原型的直接操作体验：

- 出战槽切换显式接收 `next` / `previous` 方向。跨首尾时，非目标陀螺会先在画外
  重排，不再穿过中央造成反向入场错觉；切换时不再显示“已设为出战”提示。
- 组装编辑层不再覆盖 3D 画布的指针事件。选中一个部件后，可继续点击模型其他高度
  直接切换部件；介绍卡移到左上角并轻微倾斜。
- 进入 DIY 时保留当前相机位置，再缓动到正视图；退出后按新包围盒重新拟合模型。
  DIY 改变大小和高度后，组装预览仍维持近似一致的画面占比。
- “选择竞技场”按钮缩小并靠右。地图卡片使用显式指针拖拽、单卡吸附和首尾居中；
  标准场、金属场、复合场分别增加中心导向台、同心高速导轨和分层坡面/减速块。
- 发射箭头末端增加透明大命中球，长度与 `0.35..1.0` 力度使用可逆映射。准备阶段
  陀螺与模拟均从 `(0, 4.45)` 开始，点击发射后不再横向跳位。
- 胜利结算增加独立赏金牌、三组烟花和彩带；离开战斗页会立即销毁庆祝节点。
- 根节点统一代理按钮点击音效，动态按钮同样生效；所有按钮具备黄色按压色和轻微下沉。
